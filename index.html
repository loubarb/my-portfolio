Projects

Guess the Word Game

A word guessing game using Vanilla JavaScript that allows users a limited number of chances to try and guess the hidden word. This project used Vanilla JavaScript to assess outcomes based on user input and return the appropriate data. The project also utilized Git’s version control through each step of the build.

Sticky Notes App

A sticky notes app that allows users to create & delete as many notes as they would like. This project used React to track user's saved progress when closing and reopening the app in the browser, as well as provide functionality to search for and edit notes in real-time.

Github Repo Gallery

A webpage that incorporates all of my Github repos in one place! This project used Github APIs to pull data from my Github profile and active repos and display them in an organized layout. It also includes dynamic search functionality to filter through the repos and see more details about each one.
